# Leon Agency – Minimal Creative Template

**Leon** is a modern, minimal, and fully responsive web template designed for creative agencies. It features a clean aesthetic with a focus on typography and white space, built using semantic HTML5 and modern CSS3 techniques.

## 🚀 Live Demo
[**View Live Demo**](https://amr-khaled-morsi.github.io/leon-v1/)

## ✨ Features
*   **Hero Section:** High-impact landing area with a full-width background and clear messaging.
*   **Services Grid:** A detailed breakdown of agency offerings including Web Development, Graphic Design, UI & UX, and Web Design.
*   **Portfolio Gallery:** A structured showcase for featured projects with descriptive text.
*   **About Section:** A professional layout for agency history and mission statements.
*   **Contact Section:** Clean contact area with social media integration.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Built With
*   **HTML5:** Semantic markup for better SEO and accessibility.
*   **CSS3:** Custom styling featuring CSS Grid and Flexbox for layout management.
*   **Google Fonts:** Utilizing modern web typography.
*   **Font Awesome:** For crisp, scalable iconography.
